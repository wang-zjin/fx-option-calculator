# 需求书目录

本目录存放**项目需求书**，由需求书团队根据业务需求整理，供开发团队实现与验收。

| 文档 | 说明 |
|------|------|
| [需求书-Vanilla-Web定价界面.md](./需求书-Vanilla-Web定价界面.md) | Vanilla 期权 Web 定价界面：输入/输出规格、T1/T2、Greeks、验收标准与任务拆解 |
| [需求书-组合期权定价模块.md](./需求书-组合期权定价模块.md) | 组合期权定价模块：RR（风险逆转）、海鸥期权；在 Vanilla 计算器中新增模块，可扩展其他组合 |
| [需求书-数字期权模块.md](./需求书-数字期权模块.md) | 数字期权模块：现金或无 / 资产或无，本币/外币支付；输入输出规格、计算复用、验收与任务拆解 |
| [需求书-美式期权模块.md](./需求书-美式期权模块.md) | 美式期权模块：二叉树/三叉树定价、提前行权价值、数值 Greeks、早期行权边界（看跌）；验收与任务拆解 |

新需求书命名建议：`需求书-<功能或模块名>.md`。
