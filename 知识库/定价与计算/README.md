# 定价与计算

**适用内容**：定价模型选型、公式推导、Greeks 定义与单位、数值方法（二叉树/三叉树/蒙特卡洛等）、精度与测试约定、**外汇期权本币/外币惯例**。

**主要使用**：计算模块开发、单元测试、以及与定价和 Greeks 相关的联调；前端输出标签（Premium、Vega 头寸等）的本币/外币代码也以此为准。

**权威出处（本币/外币惯例）**：  
- **Foreign Exchange Option Pricing A Practitioners Guide.pdf**（本文件夹内）— 书中对 domestic/foreign、报价惯例有定义；  
- **外汇期权-本币外币惯例与出处.md**（本文件夹内）— 知识库团队必读：从 PDF 摘录本币/外币约定并维护，供项目实现与新增货币对时对照。

**调取示例**：
- *「请结合 `知识库/定价与计算/` 中的模型说明，实现/校验 Garman-Kohlhagen 的 d1/d2 与期权价格。」*
- *「在知识库定价与计算中查找 Greeks 的定义与单位，确保输出一致。」*
- *「本币/外币惯例见 `外汇期权-本币外币惯例与出处.md` 及 PDF *Foreign Exchange Option Pricing A Practitioners Guide*。」*

**可放置**：模型选型文档、公式推导、Greeks 定义、数值方法说明、标准案例与基准值、本币/外币惯例摘录等。
